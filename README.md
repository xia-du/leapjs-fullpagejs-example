# LeapJS + fullPage.js Example
This is a simple example about how to navigate a web page with hand gestures.

You need to [setup your Leap Motion Controller][99d4177e] to run the test.

[Live Demo][42ad1bae]

## Dependencies
- [LeapJS][2999ef86]
- [fullPage.js][6a904870]
